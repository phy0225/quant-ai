## 一、行情数据

**使用方**：技术分析师、情绪分析师
**数据来源**：AKShare → 东方财富，日线 K 线，近 120 天
**缓存策略**：本地文件缓存 4 小时
**接口**：`ak.stock_zh_a_hist(symbol, period="daily", adjust="qfq")`

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `current_price` | 最新收盘价 | 元 | 技术规则判断、持仓浮盈计算 |
| `prev_close` | 昨收价 | 元 | 涨跌幅基准 |
| `change_pct` | 今日涨跌幅 | % | 情绪分析参考 |
| `high_52w` | 52 周最高价 | 元 | 年内高点参考 |
| `low_52w` | 52 周最低价 | 元 | 年内低点参考 |
| `ma5` | 5 日均线 | 元 | 短期趋势 |
| `ma10` | 10 日均线 | 元 | 短期趋势 |
| `ma20` | 20 日均线 | 元 | 中期趋势（多空排列核心） |
| `ma60` | 60 日均线 | 元 | 长期趋势（多空排列核心） |
| `rsi` | RSI(14) 相对强弱指数 | 0-100 | **超买超卖判断（技术规则核心）** |
| `macd` | MACD 快慢线差值 | 元 | 金叉/死叉判断 |
| `macd_signal` | MACD 信号线 | 元 | 金叉/死叉判断 |
| `bb_upper` | 布林带上轨（MA20+2σ） | 元 | 价格突破判断 |
| `bb_mid` | 布林带中轨（MA20） | 元 | 价格中枢 |
| `bb_lower` | 布林带下轨（MA20-2σ） | 元 | 价格突破判断 |
| `volume_today` | 今日成交量 | 手 | 量能参考 |
| `volume_avg_20d` | 20 日平均成交量 | 手 | 成交量基准 |
| `volume_ratio` | 量比（今日/20日均量） | 倍 | 放量/缩量判断（>1.5 倍确认趋势） |
| `trend` | 趋势标签 | 多头/空头/震荡 | 摘要生成参考 |
| `recent_closes` | 近 5 日收盘价 | 元数组 | LLM 上下文 |
| `recent_dates` | 近 5 日日期 | 日期数组 | LLM 上下文 |

## 二、基本面数据

**使用方**：基本面分析师
**数据来源**：AKShare → 东方财富 / 上市公司财报
**缓存策略**：本地文件缓存 7 天（季报频率低）
**主要接口**：
- `ak.stock_zh_a_spot_em()` — 实时行情（含估值字段）
- `ak.stock_financial_abstract(symbol=)` — 财务摘要（转置表格格式）
- `ak.stock_zh_a_indicator(symbol=, timeframe="全部")` — 每股指标

### 估值数据

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `stock_name` | 股票中文简称 | 文字 | 显示 |
| `pe_ratio` | 市盈率 PE（动态） | 倍 | 估值高低判断 |
| `pb_ratio` | 市净率 PB | 倍 | 净资产溢价判断 |
| `total_mv` | 总市值 | 亿元 | 公司规模 |
| `turnover_rate` | 换手率 | % | 流动性参考 |

### 成长性数据

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `revenue_yoy` | 营收同比增速 | % | **成长性核心判断** |
| `net_profit_yoy` | 净利润同比增速 | % | 盈利成长性 |
| `eps` | 每股收益 EPS | 元/股 | 盈利参考 |

### 盈利质量数据

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `roe` | 净资产收益率 ROE | % | **盈利质量核心指标** |
| `gross_margin` | 销售毛利率 | % | 产品/服务竞争力 |
| `net_margin` | 销售净利率 | % | 最终盈利水平 |

### 财务健康数据

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `debt_ratio` | 资产负债率 | % | 财务杠杆风险 |
| `report_period` | 最新报告期 | 如 2024-Q3 | 数据时效标注 |

### 数据级别说明

| 级别 | 条件 | 置信度上限 | 说明 |
|------|------|-----------|------|
| Level A | 估值数据 + 财务数据均有 | 0.80 | 完整分析 |
| Level B | 仅有估值 或 仅有财务（任一） | 0.60 | 有限分析，摘要标注 |
| Level C | 所有字段均为空 | 0.25 | **不调用 LLM，直接返回 hold，不参与投票** |

## 三、新闻数据

**使用方**：新闻分析师
**数据来源**：AKShare → 东方财富个股新闻 + 财联社电报
**缓存策略**：本地文件缓存 2 小时
**主要接口**：
- `ak.stock_news_em(symbol=)` — 东方财富个股新闻
- `ak.stock_telegraph_cls()` 等（多接口兜底）— 财联社宏观电报

### 个股新闻

| 字段名 | 中文说明 | 用途 |
|--------|---------|------|
| `title` | 新闻标题 | LLM 情感分析（**主要依据**） |
| `source` | 新闻来源 | 可信度参考（如证券时报、Reuters） |
| `published` | 发布日期 | 时效性判断 |
| `content` | 正文摘要（前 200 字） | LLM 深度理解 |

### 宏观快讯（财联社电报）

| 字段名 | 中文说明 | 用途 |
|--------|---------|------|
| `title` | 电报标题 | 宏观背景参考 |
| `published` | 发布时间 | 时效性 |
| `content` | 内容摘要 | LLM 背景理解 |

### 数据级别说明

| 级别 | 条件 | 置信度上限 |
|------|------|-----------|
| Level A | ≥ 3 条近期新闻 | 0.75 |
| Level B | 1-2 条新闻 | 0.55 |
| Level C | 无新闻 | 0.20，**不调 LLM，不参与投票** |

## 四、情绪数据

**使用方**：情绪分析师
**数据来源**：AKShare → 东方财富 / 交易所
**缓存策略**：本地文件缓存 2 小时

### 个股情绪

| 字段名 | 中文说明 | 单位 | 接口 | 评分权重 |
|--------|---------|------|------|---------|
| `score` | 东财个股综合评分 | 0-100 | `ak.stock_comment_em()` | >65 得 +1.5 分；<40 得 -1.0 分 |
| `win_rate` | 胜率 | % | 同上 | 辅助参考 |

### 市场情绪（全市场）

| 字段名 | 中文说明 | 单位 | 接口 | 评分权重 |
|--------|---------|------|------|---------|
| `total_flow` | 北向资金今日净流入 | 亿元 | `ak.stock_em_hsgt_north_net_flow_in()` 等 | >30亿 得 +1.0 分；<-30亿 得 -1.0 分 |
| `limit_up_count` | 今日涨停家数 | 只 | `ak.stock_zt_pool_em()` 等 | 影响情绪比 |
| `limit_down_count` | 今日跌停家数 | 只 | `ak.stock_dt_pool_em()` 等 | 影响情绪比 |
| `sentiment_ratio` | 情绪比（涨停/涨跌停合计） | 0-1 | 计算得出 | >0.7 得 +0.5 分；<0.3 得 -0.5 分 |

### 技术派生情绪（从行情数据计算）

| 字段名 | 中文说明 | 评分权重 |
|--------|---------|---------|
| `rsi` | RSI(14) | >72 得 -2.0 分（超买看空）；<28 得 +2.0 分（超卖看多） |
| `volume_ratio` | 成交量比 | 辅助参考，配合方向加权 |

### 综合评分机制

```
总分 = 东财评分贡献 + RSI贡献 + 北向资金贡献 + 涨跌停比贡献

总分 ≥ +2.0 → buy，置信度 = min(0.50 + 总分×0.05, 0.78)
总分 ≤ -2.0 → sell，置信度 = min(0.50 + |总分|×0.05, 0.75)
其余        → hold，置信度 = 0.48
```

## 五、持仓数据

**使用方**：所有 Agent（触发分析时自动注入）
**数据来源**：本地数据库 `portfolio_holdings` 表，由用户维护或审批通过后自动更新
**缓存策略**：无缓存，每次触发分析实时读取

| 字段名 | 中文说明 | 单位 | 用途 |
|--------|---------|------|------|
| `weight` | 当前仓位权重 | 0-1 | 仓位上限风控检查；推荐调仓计算 |
| `cost_price` | 建仓成本价 | 元 | **浮盈/亏计算；触发止盈止损规则** |
| `current_price` | 最新市价（刷新后） | 元 | 实时浮盈计算 |
| `pnl_pct` | 浮动盈亏 | % | 覆盖技术/情绪信号（见下表） |
| `quantity` | 持股数量 | 股 | 市值计算 |
| `market_value` | 持仓市值 | 元 | 组合汇总 |
| `symbol_name` | 股票简称 | 文字 | 摘要生成 |

### 持仓覆盖规则

| 条件 | 被覆盖的信号 | 覆盖效果 |
|------|------------|---------|
| 浮盈 ≥ 80% | 技术买入信号 | 强制改为 hold，置信度 0.50 |
| 浮盈 ≥ 50% | 技术买入/持有 | 置信度上限压至 0.55，提示止盈 |
| 浮亏 ≥ 30% | 技术持有信号 | 置信度上限压至 0.50，提示止损 |
| 浮盈 ≥ 50% | 情绪买入信号 | 方向改为 hold，提示抛售压力 |
| 浮亏 ≥ 25% | 情绪卖出信号 | 方向改为 hold，提示恐慌已释放 |

## 六、风控配置数据

**使用方**：风控管理员（参与 pipeline 决策）
**数据来源**：本地数据库 `risk_configs` 表
**缓存策略**：无缓存，每次触发实时读取

| 字段名 | 中文说明 | 默认值 | 用途 |
|--------|---------|-------|------|
| `max_position_weight` | 单标的仓位上限 | 20% | 超过上限时风险等级为 high，触发 hold |
| `daily_loss_warning_threshold` | 日内亏损警告线 | 3% | 触发 L1 熔断 |
| `daily_loss_suspend_threshold` | 日内亏损暂停线 | 6% | 触发 L2 熔断，审批被拒 |
| `max_drawdown_emergency` | 最大回撤紧急线 | 15% | 触发 L3 紧急停止 |
| `circuit_breaker_level` | 当前熔断等级 | 0（正常） | L2+ 时拒绝所有触发和审批 |
| `emergency_stop_active` | 紧急停止状态 | false | 激活后拒绝所有操作 |

## 七、数据降级汇总

| Agent | 数据 | 缺失原因 | 降级处理 |
|-------|------|---------|---------|
| 技术分析师 | 行情 K 线 | AKShare 网络超时 | 返回 hold 0.40，不参与投票 |
| 基本面分析师 | 财务数据 | 接口名变动 / 网络问题 | Level B（置信度≤0.60）或 Level C（拒绝方向判断） |
| 新闻分析师 | 个股新闻 | 接口失败 / 无近期新闻 | Level C（置信度 0.20，不参与投票） |
| 情绪分析师 | 东财评分 | 接口参数变动 | 该项得分记 0，其余指标正常 |
| 情绪分析师 | 北向资金 | 接口名变动 | 该项得分记 0，不影响其他指标 |
| 情绪分析师 | 涨跌停数据 | 非交易时段 / 接口失败 | 该项得分记 0 |
